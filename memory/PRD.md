# PRD - Landing Page Fernanda Loyola - Fonoaudiologia Infantil

## Problem Statement
Landing page profissional para fonoaudióloga infantil Fernanda Loyola, com foco em conversão via WhatsApp para campanhas do Google Ads.

## User Persona
- Mães e responsáveis de crianças com necessidades de fonoaudiologia
- Região: Baixada Fluminense / Nova Iguaçu - RJ

## Core Requirements
- Página 100% responsiva e mobile-first
- Foco em conversão para WhatsApp
- Visual acolhedor, profissional e humanizado
- Paleta: roxo, lilás, verde, rosa, bege
- SEO básico para Google Ads
- Sem pop-ups, sem promessas exageradas

## What's Been Implemented (Dez 2025)
- Hero Section com foto da profissional, nome, título, subtítulo, CTA WhatsApp
- Seção "Para Quem É" com 3 cards em cores pastel (lilás, rosa, verde)
- Seção "Sobre" com bio profissional completa (15 anos, 3 pós-graduações)
- Seção "Como Funciona" com 3 etapas + imagem lifestyle
- Seção "Atendimento Presencial" com localização (Baixada Fluminense / Nova Iguaçu)
- CTA Final com botão WhatsApp
- Footer com Instagram, telefone, copyright
- Botão flutuante de WhatsApp com animação pulse
- Tipografia: Cormorant Garamond (títulos) + Manrope (corpo)
- Meta tags SEO otimizadas
- Todos os links WhatsApp com mensagem pré-configurada
